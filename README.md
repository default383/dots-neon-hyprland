DEPENDENCIAS

-wofi
-hyprpaper
-grim
-waybar
-wlogout
-hyprlock
-hypridle
-blueman
-nm(network manager)
-kitty
-thunar
zsh + oh-my-zsh + tema neón (no es obligatorio, pero sumamente recomendado)

Una vez tengan los archivos descargados, deben moverlo a ~/.config y a sus respectivas carpetas. Si no existen, creenlas.
Para hyprpaper, este lee la carpeta ~/Imágenes/hyprpaper para sus fondos de pantalla, por lo que deben poner su imágen ahí.


COMBINANCIONES DE TECLAS:

Redimensionar las ventanas "SUPER + ALT + click-izquierdo"
Mover ventanas "SUPER + click-izquierdo"
Activar hyprlock "SUPER + L"
Cerrar sesión "SUPER + M"
Captura de pantalla "SUPER + PRINT"
Abrir firefox "SUPER + W"
Abrir kitty "SUPER + Q"
Abrir Thunar "SUPER + E"
Abrir lista de aplicaciones "SUPER + R"

A TENER EN CUENTA:

Usaba gnome con el tema candy, por lo que si tienes GNOME en Fedora por defecto, puedes descargarte retoques, extensiones y gestor de extensiones más el tema "candy" para mejores resultados. Si no entiendes del tema, puedes buscar en youtube sobre cómo añadir temas a gnome.
